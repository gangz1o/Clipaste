## ✨ 版本更新说明 / Release Notes

**1. 📏 纵向紧凑布局**
- 新增“纵向紧凑”模式，大幅提升空间利用率。
- 支持通过快捷键在横向、纵向、紧凑三种布局间快速循环切换。

**2. 👁️ 侧边预览面板**
- 为纵向布局新增侧边预览窗格。
- 悬停或选中条目时，可即时查看图片大图、代码高亮及完整长文本。

**3. 📦 Maccy 历史导入**
- 支持从 Maccy 迁移数据。
- 方便新用户快速导入原有剪贴板历史记录。

**4. 🛡️ 防误删保护**
- 新增“Cmd + Backspace”删除选项。
- 开启后可有效避免在操作中意外删除重要的剪贴板记录。

---

**1. 📏 Vertical Compact Layout**
- Introduced a space-efficient "Vertical Compact" mode to show more items at once.
- Supports cycling through all layouts via shortcut: Horizontal → Vertical → Compact.

**2. 👁️ Preview Panel**
- Added an optional side panel for vertical layouts.
- Real-time preview for images, syntax-highlighted code, and full text on hover/selection.

**3. 📦 Maccy History Import**
- Now supports migrating clipboard history from Maccy.
- Found in Settings → Import History → Maccy.

**4. 🛡️ Deletion Safety**
- Added an optional setting requiring "Cmd + Backspace" for deletion.
- Provides extra protection against accidental data loss.
