### ✨ 本次更新

🚀 功能与体验增强

- **全新链接展示模式**：新增「丰富模式」（默认开启），直观呈现网页标题、图标及域名信息。如果您更偏爱极简外观，也可在「高级 - 链接显示模式」中随时切换。
- **链接快捷操作**：为 URL 卡片新增了右键菜单，现在可以直接呼出菜单并在默认浏览器中快速打开该链接。

🐛 修复与优化

- **Dock 栏图标逻辑**：修复了应用后台运行时，图标仍然异常常驻 macOS Dock 栏的问题，让应用更纯粹地在状态栏默默工作。
- **首次启动体验修复**：修复了全新安装后，首次同时打开剪切板与设置页时，点击设置页面会导致窗口意外隐藏的 Bug。
- **国际化翻译补全**：完善了部分界面和极端场景下的多语言显示，提升不同语言环境下的使用体验。

### ✨ What's New

🚀 Features & Enhancements

- **New URL Display Modes**: Introducing "Rich Mode" (enabled by default) to intuitively display web titles, favicons, and domains. You can easily switch back to the minimal mode in Advanced Settings.
- **URL Context Menu**: Added a handy right-click menu to URL cards, allowing you to quickly open links directly in your default browser.

🐛 Bug Fixes & Improvements

- **Dock Icon Behavior**: Fixed an issue where the app icon would persistently and incorrectly remain in the macOS Dock.
- **First-Launch Experience**: Resolved a bug that caused the window to unexpectedly hide when interacting with the settings page immediately after a fresh installation.
- **Localization Updates**: Completed missing multi-language translations across various scenarios to improve the experience for global users.
